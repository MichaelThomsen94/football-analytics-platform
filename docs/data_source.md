# Data Source

## Dataset

European Soccer Database

Source:
https://www.kaggle.com/datasets/hugomathien/soccer

## Description

European football database containing:

* Matches
* Players
* Teams
* Leagues
* Countries
* Player attributes
* Team attributes

Coverage:

* 11 European countries
* Seasons 2008-2016
* 25,000+ matches
* 10,000+ players

## Project Goal

Build a Football Analytics Platform using Microsoft Fabric and Power BI.
